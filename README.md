# Практична робота "Масиви, вирази, керування виконанням програми"

## Завдання: №15

### Опис завдання

На складі розташовано 50,000 полиць. Через технічну несправність друкарського верстата цифра 2 не була надрукована. Необхідно підрахувати кількість номерів у діапазоні від 1 до 50,000, які містять хоча б одну двійку і, відповідно, є бракованими.
### Деталі реалізації

Програма не використовує масиви або рядки для аналізу чисел. Перевірка відбувається шляхом ділення числа на 10 у циклі while.
Використано тип int для лічильників, оскільки кількість полиць (50,000) та результат (23,756) вкладаються в діапазон значень цього типу, забезпечуючи максимальну швидкість обробки.

### Результат обчислення

Для 50,000 полиць кількість бракованих табличок становить: 23,756.
